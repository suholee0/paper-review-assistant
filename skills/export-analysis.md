# Export Analysis

채팅 내용을 바탕으로 기존 분석 문서(analysis.md)를 보강하는 skill.

## 입력

- `{{paperDir}}` — 논문 디렉토리 경로
- `{{chatHistory}}` — 사용자와 AI 간의 채팅 기록 (JSON)

## 작업 흐름

1. `{{paperDir}}/analysis.md` 파일을 읽는다
2. 채팅 기록에서 새로운 인사이트를 추출한다:
   - 사용자가 질문한 개념에 대한 상세 설명
   - AI가 추가로 제공한 분석 내용
   - 사용자가 특히 관심을 가진 부분
3. 기존 분석 문서의 구조를 유지하면서 채팅에서 얻은 인사이트를 통합한다:
   - 기존 섹션에 보충 설명 추가
   - 새로운 "Discussion Notes" 섹션 추가 (채팅에서 나온 주요 Q&A 정리)
4. 보강된 문서를 `{{paperDir}}/analysis.md`에 저장한다

## 출력 형식

마크다운 문서. 기존 analysis.md 구조를 유지하되, 다음을 추가:

- 기존 섹션 내에 `> 💬` blockquote로 채팅에서 나온 보충 설명 삽입
- 문서 끝에 `## Discussion Notes` 섹션 추가:
  - 채팅에서 나온 핵심 Q&A를 정리
  - 사용자가 특히 관심 가진 토픽 요약

## 주의사항

- 기존 내용을 삭제하지 않는다
- 채팅 내용이 이미 분석에 포함된 경우 중복 추가하지 않는다
- 한국어로 작성한다
